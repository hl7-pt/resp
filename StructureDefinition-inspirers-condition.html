Resource Profile: Inspirers Condition Profile

Official URL: http://example.com/fhir/example/StructureDefinition/inspirers-condition Version: 1.0.0
Active as of 2024-09-24 Computable Name: InspirersCondition

Defines constraints and extensions on the Condition resource for the minimal set of data to query and retrieve patient observations.
